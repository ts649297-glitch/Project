# Task 1 - Basic HTML Structure

A simple HTML project that displays a "Hello World" message.

This file demonstrates the basic skeleton of an HTML document, including the Doctype declaration, head, and body tags.


1. Save the code as `index.html`.
2. Open `index.html` in any web browser.

* **Document Type:** Declared as HTML5.
* **Content:** Displays "HELLO WORLD !" using a paragraph (`<p>`) tag.
